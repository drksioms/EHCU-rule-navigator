# EHCU Rule Navigator

## Version

Version 1.2

## 目的

EHCU Rule Navigator は、済生会横浜市東部病院 救命救急センターにおける EHCU 入室時・転科時・Dr.call の院内運用を確認するための静的Webアプリです。

若手医師・当直医が、誰へ連絡するか、どの順番で処理するか、次に何をするかを迷わず確認できることを目的とします。

このアプリは患者情報を入力・保存しません。医療判断そのものを自動化するものではなく、院内運用ルール確認のみを目的とします。

## トップ画面

トップ画面には以下の4項目を表示します。

1. EHCU入室時フロー
2. 転科時フロー
3. EHCU看護師向けDr.callフロー
4. ルール参照

## 基本方針

- 患者情報は入力しない
- 院内運用専用
- 単一HTMLで完結
- スマホ利用前提
- 大量選択肢はプルダウン
- 少ない選択肢はボタン
- チェックボックスは使わない
- 判定結果を上、次の操作を下に表示
- ベースカラーは濃い青
- Apple風のシンプルで洗練されたUI
- 医療現場で瞬時に理解できる視認性を優先

## 重要な文言統一

- EICU関連の選択肢は「EICU加療歴なし」「EICU加療歴あり」の2つに統一
- 注意文は「現在地ではなく、EICU加療歴で判定する」に統一
- 救急科とその他は別ルートとして扱う
- 泌尿器科・血管外科・心臓血管外科への夜間・休日転科は行わない

## 総合内科対象疾患

総合内科対象疾患はプルダウンで表示します。

- 誤嚥性肺炎
- 尿路感染症
- 胃腸炎（細菌性・感染性）
- 敗血症・菌血症
- 社会的入院
- 膠原病
- 血液疾患
- 貧血（消化管出血を除く）
- アレルギー性疾患
- 蜂窩織炎・うっ滞性皮膚炎
- 横紋筋融解
- 該当なし

「該当なし」以外は総合内科対象疾患あり、「該当なし」は総合内科対象疾患なしとして扱います。

## 主科・転科先

主科・転科先はプルダウンで表示します。

- 循環器内科
- 脳神経内科
- 消化器内科
- 腎臓内科
- 呼吸器内科
- 糖尿病内分泌内科
- 脳神経外科
- 消化器外科
- 泌尿器科
- 血管外科
- 心臓血管外科
- 救急科
- その他

救急科は、最初から救急科主科としてEHCU入室するルートです。その他は、選択肢に該当する主科がないため、いったん救急科主科でEHCU入室し、転科時は転科時フローに従うルートです。

## 内科輪番当番科として選択可能な診療科

- 循環器内科
- 脳神経内科
- 消化器内科
- 腎臓内科
- 呼吸器内科
- 糖尿病内分泌内科

## EHCU入室時フロー概要

1. NEWS判定を行う
2. NEWS＜4 は一般病棟をまず考慮する
3. NEWS≧4 はEHCU入室を検討し、総合内科対象疾患を選択する
4. 総合内科対象疾患ありは総合内科ルートへ進む
5. 平日日勤の総合内科ルートでは、井本先生へ連絡し、総合内科主科または内科輪番当番科主科の分岐を確認する
6. 夜間・休日の総合内科ルートでは、EHCU病棟番へ連絡し、総合内科主科でEHCU入室する
7. 総合内科対象疾患なしは主科を選択し、平日日勤または夜間・休日の他科入室フローへ進む
8. 救急科とその他は別ルートとして扱う

## 転科時フロー概要

1. 総合内科対象疾患を選択する
2. 総合内科対象疾患ありは総合内科転科ルートへ進む
3. 総合内科転科ルートでは、時間帯とEICU加療歴を確認する
4. 平日日勤かつEICU加療歴なしは井本先生へ転科依頼する
5. EICU加療歴ありは内科輪番当番へ直接依頼、または夜間・休日では内科輪番確認後に内科診療科へ依頼する
6. 総合内科対象疾患なしは転科先を選択する
7. その他はEHCU内転科対象外として転棟時転科ルートへ進む
8. 泌尿器科・血管外科・心臓血管外科への夜間・休日転科は行わない

## EHCU看護師向けDr.callフロー概要

EHCU看護師向けDr.callフローでは、まず時間帯を選択し、次に主科または転科先を選択して、誰をDr.call対象とするかを確認します。

EHCU入室時とEHCU内転科後で異なる周辺作業は、判定結果の下に作業メモとして分けて表示します。

- 平日日勤の総合内科：総合内科医師・NP
- 平日日勤の救急科：ER当直医師
- 平日日勤のその他診療科：Call指示に従う
- 夜間休日の総合内科・消化器内科・腎臓内科・呼吸器内科・糖尿病内分泌内科：EHCU病棟番医師
- 夜間休日の循環器内科：Heart line当直医師
- 夜間休日の脳神経内科・脳神経外科：Stroke line当直医師
- 夜間休日の救急科：ER当直医師
- 泌尿器科・血管外科・心臓血管外科への夜間休日EHCU内転科は対象外

## 転棟時転科ルート概要

- 内因性疾患は内科輪番の順番通り振り分け
- 外科系・マイナー疾患はそれぞれの科へ相談
- 転棟可能となった時点で、転棟後主科へ連絡
- 直接電話またはLINE WORKSで情報共有
- 夜間・休日の転棟は原則控える
