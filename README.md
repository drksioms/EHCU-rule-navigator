# EHCU Rule Navigator

## 概要

EHCU Rule Navigator は、済生会横浜市東部病院 救命救急センターにおける EHCU 入室時・転科時の院内運用を確認するための静的Webアプリです。

若手医師・当直医が、以下を迷わず確認できることを目的とします。

- 誰へ連絡するか
- どの順番で処理するか
- 次に何をするか
- EHCU内転科が可能か
- 転棟時転科ルートへ進むべきか

このアプリは患者情報を入力・保存しません。  
院内運用ルール確認のみを目的とします。

## 基本方針

- 患者情報は入力しない
- 院内運用専用
- 単一HTMLで完結
- スマホ利用前提
- EHCU入室時フローと転科時フローを最初に選ぶ
- 大量の選択肢だけプルダウンにする
- 少ない選択肢はボタンにする
- チェックボックスは使わない
- 判定結果を上に表示する
- 次の操作を判定結果の下に表示する
- 実行STEPを大きく見やすく表示する
- 片手操作しやすいUIにする
- 誤タップ防止のため、ボタン間隔を十分に空ける
- ベースカラーは濃い青
- Appleの現代的UIに近い、シンプルで洗練された見た目にする
- 余計な装飾は入れない
- 医療現場で瞬時に理解できる視認性を優先する

## 総合内科対象疾患

総合内科ルートに進めるのは、以下の疾患・病態を選択した場合です。

- 誤嚥性肺炎
- 尿路感染症
- 胃腸炎
- 敗血症・菌血症
- 社会的入院
- 膠原病
- 血液疾患
- 貧血
- アレルギー性疾患
- 蜂窩織炎・うっ滞性皮膚炎
- 横紋筋融解
- 該当なし

大量選択肢のため、アプリ上ではプルダウンで表示します。

# EHCU内転科成立時の共通STEP

EHCU内転科が成立した場合は、総合内科ルート・他科ルートを問わず、以下を表示します。

1. 転科先へ依頼
2. 直接電話またはLINE WORKSで情報共有
3. 救急科で電子カルテ上の主科・主治医変更
4. 救急科で重症指示簿を消去
5. 平日日勤：転科先が一般病棟用指示簿へ書き換え
6. 夜間・休日：EHCU病棟番が一般病棟用指示簿へ書き換え
7. 平日日勤：転科先がドクターコール順序を記載
8. 夜間・休日：翌平日日勤帯に転科先がドクターコール順序を記載
