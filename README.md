# EHCU Rule Navigator

## Version

Version 1.4

## 目的

EHCU Rule Navigator は、済生会横浜市東部病院 救命救急センターにおける EHCU 入室時・転科時・Dr.call の院内運用を確認するための静的Webアプリです。

若手医師・当直医が、誰へ連絡するか、どの順番で処理するか、次に何をするかを迷わず確認できることを目的とします。

このアプリは患者情報を入力・保存しません。医療判断そのものを自動化するものではなく、院内運用ルール確認のみを目的とします。

トップ画面には以下の5項目を表示します。

## 基本方針

- 患者情報は入力しない
- 院内運用専用
- 単一HTMLで完結
- スマホ利用前提
- 大量選択肢はプルダウン
- 少ない選択肢はボタン
- チェックボックスは使わない
- 判定結果を上、次の操作を下に表示
- ベースカラーは濃い青
- Apple風のシンプルで洗練されたUI
- 医療現場で瞬時に理解できる視認性を優先

## 重要な文言統一

- 総合内科対象疾患はICU加療歴を問わない
- 平日日勤の救急科から総合内科へ転科時は井本先生にまず相談
- 夜間休日は相談不要で総合内科転科OK
- EICUから総合内科へ転科時、総合内科で対応困難な場合は内科輪番で主科決定
- 救急科とその他は別ルートとして扱う
- 泌尿器科・血管外科・心臓血管外科への夜間・休日転科は行わない

## 総合内科対象疾患

総合内科対象疾患はプルダウンで表示します。

- 非心原性心停止蘇生後（主科割り振り不可）
- 誤嚥性肺炎
- 尿路感染症
- 胃腸炎（細菌性・感染性）
- 敗血症・菌血症
- 社会的入院
- 膠原病
- 血液疾患
- 貧血（消化管出血を除く）
- アレルギー性疾患
- 蜂窩織炎・うっ滞性皮膚炎
- 横紋筋融解

「該当なし」以外は総合内科対象疾患あり、「該当なし」は総合内科対象疾患なしとして扱います。

総合内科対象疾患はICU加療歴を問いません。

ただし、EHCU入室時フローのプルダウンには「非心原性心停止蘇生後（主科割り振り不可）」を含めません。

## 総合内科主科

- EHCU入室時：総合内科対象疾患であれば、総合内科主科・井本先生主治医で入院
- EHCU入室時は相談不要
- EHCU総合内科主科入室時も、EICU当番医師へ担当医より入室連絡を行う
- 平日日勤の救急科から転科時：総合内科対象疾患であれば井本先生にまず相談
- 夜間休日：相談不要で総合内科転科OK
- EICUから転科時：総合内科で対応困難な場合は内科輪番で主科決定
- Call 平日：NP・井本先生
- Call 夜間休日：内科病棟番（現EHCU当直）

## EHCU入室時フロー概要

1. NEWS判定を行う
2. NEWS＜4 は一般病棟をまず考慮する
3. NEWS≧4 はEHCU入室を検討し、総合内科対象疾患を選択する
4. 総合内科対象疾患ありは総合内科ルートへ進む
5. 総合内科対象疾患であれば、総合内科主科・井本先生主治医で入院する
6. 入院時は相談不要
7. EICU当番医師へ担当医より入室連絡を行う
8. 総合内科対象疾患なしは主科を選択し、平日日勤または夜間・休日の他科入室フローへ進む
9. 救急科とその他は別ルートとして扱う

## 転科時フロー概要

1. 総合内科対象疾患を選択する
2. 総合内科対象疾患ありは総合内科転科ルートへ進む
3. 総合内科対象疾患はICU加療歴を問わない
4. 平日日勤の救急科から転科時は井本先生にまず相談する
5. 夜間休日は相談不要で総合内科転科OK
6. EICUから転科時に総合内科で対応困難な場合は、内科輪番で主科決定する
7. 総合内科対象疾患なしは転科先を選択する
8. その他はEHCU内転科対象外として転棟時転科ルートへ進む
9. 泌尿器科・血管外科・心臓血管外科への夜間・休日転科は行わない

## EHCU看護師向けDr.callフロー概要

EHCU看護師向けDr.callフローでは、まず時間帯を選択し、次に主科または転科先を選択して、誰をDr.call対象とするかを確認します。

EHCU入室時とEHCU内転科後で異なる周辺作業は、判定結果の下に作業メモとして分けて表示します。

- 平日日勤の総合内科：NP・井本先生
- 平日日勤の救急科：ER当直医師
- 平日日勤のその他診療科：Call指示に従う
- 夜間休日の総合内科・消化器内科・腎臓内科・呼吸器内科・糖尿病内分泌内科：内科病棟番（現EHCU当直）
- 夜間休日の循環器内科：Heart line当直医師
- 夜間休日の脳神経内科・脳神経外科：Stroke line当直医師
- 夜間休日の救急科：ER当直医師
- 泌尿器科・血管外科・心臓血管外科への夜間休日EHCU内転科は対象外

## EHCU入院患者容態変化時フロー概要

EHCU入院患者の容態変化時に、NEWS score確認からDr.call、EICU/GICUへのStep up検討までを確認します。

1. NEWS scoreを確認
2. 黄色・赤に該当する場合は、1st Dr.call + EICU当番医call
3. EICU当番医対応困難な時は、GICU Dr.call
4. EICU/GICUへのStep upを検討

## 転棟時転科ルート概要

- 内因性疾患は内科輪番の順番通り振り分け
- 外科系・マイナー疾患はそれぞれの科へ相談
- 転棟可能となった時点で、転棟後主科へ連絡
- 直接電話またはLINE WORKSで情報共有
- 夜間・休日の転棟は原則控える
