# AGENTS.md

## このプロジェクトの目的

このプロジェクトは、済生会横浜市東部病院 救命救急センターにおける EHCU 入室時・転科時・Dr.call の院内運用を、若手医師・当直医が迷わず確認できる静的Webアプリとして実装するためのものです。

アプリ名は EHCU Rule Navigator です。

## 最重要方針

このアプリは医療判断そのものを自動化するものではありません。

目的は、院内運用ルールをSTEP形式で見やすく提示し、連絡先・処理順序・次に行う行動を確認しやすくすることです。

患者情報は絶対に入力させないでください。

今後の修正では Version 1.2 のルールを優先してください。

- 表示バージョンは `Version 1.2`
- ヘッダーには、済生会横浜市東部病院、救命救急センター、EHCU Rule Navigator、Version 1.2 を表示
- フッターには `Created by Kosei Omasa` を表示
- トップ画面には `EHCU入室時フロー`、`転科時フロー`、`EHCU看護師向けDr.callフロー`、`ルール参照` を表示
- EHCU看護師向けDr.callフローをトップ画面から削除しない

## 実装方針

以下を厳守してください。

- 単一HTMLで作成する
- ファイル名は `index.html`
- HTML / CSS / JavaScript をすべて `index.html` 内に書く
- Reactは使わない
- Tailwindは使わない
- 外部ライブラリは使わない
- CDNは使わない
- localStorageは使わない
- 患者情報入力欄は作らない
- フォーム送信機能は作らない
- データ保存機能は作らない
- サーバー通信は行わない
- GitHub Pagesでそのまま公開できる静的HTMLにする
- ブラウザで直接 `index.html` を開いて動作するようにする

## UI方針

- スマホ利用を最優先する
- 片手操作しやすいUIにする
- 誤タップ防止のため、ボタン間隔を十分に空ける
- 大量選択肢はプルダウンで表示する
- 少ない選択肢はボタンで表示する
- チェックボックスは使わない
- 判定結果を上、次の操作を下に表示する
- 実行STEPを大きく見やすく表示する
- ベースカラーは濃い青
- Apple風のシンプルで洗練されたUIにする
- 余計な装飾や派手なアニメーションは入れない
- 医療現場で瞬時に理解できる視認性を優先する

## 共通選択肢

### 総合内科対象疾患

プルダウンで表示する。

- 誤嚥性肺炎
- 尿路感染症
- 胃腸炎（細菌性・感染性）
- 敗血症・菌血症
- 社会的入院
- 膠原病
- 血液疾患
- 貧血（消化管出血を除く）
- アレルギー性疾患
- 蜂窩織炎・うっ滞性皮膚炎
- 横紋筋融解
- 該当なし

判定：

- `該当なし` 以外：総合内科対象疾患あり
- `該当なし`：総合内科対象疾患なし

### 主科・転科先

プルダウンで表示する。

- 循環器内科
- 脳神経内科
- 消化器内科
- 腎臓内科
- 呼吸器内科
- 糖尿病内分泌内科
- 脳神経外科
- 消化器外科
- 泌尿器科
- 血管外科
- 心臓血管外科
- 救急科
- その他

救急科とその他は別ルートです。

- 救急科：最初から救急科主科としてEHCU入室するルート
- その他：選択肢に該当する主科がないため、いったん救急科主科でEHCU入室し、転科時は転科時フローに従うルート

### 内科輪番当番科として選択可能な診療科

- 循環器内科
- 脳神経内科
- 消化器内科
- 腎臓内科
- 呼吸器内科
- 糖尿病内分泌内科

## Version 1.2フロー要点

### EHCU入室時フロー

- NEWS判定はボタンで表示する
- NEWS＜4 は一般病棟をまず考慮する
- NEWS≧4 はEHCU入室を検討し、総合内科対象疾患選択へ進む
- 総合内科対象疾患ありは総合内科ルート
- 平日日勤の総合内科ルートでは井本先生へ連絡し、総合内科主科で承諾、または内科輪番当番科主科の分岐を表示する
- 夜間・休日の総合内科ルートではEHCU病棟番へ連絡し、Dr.callはEHCU病棟番医師
- 総合内科対象疾患なしでは主科を選択する
- 救急科主科のDr.callはER当直医師
- その他は救急科主科でEHCU入室し、転科時は転科時フローに従う

### 転科時フロー

- 総合内科対象疾患をプルダウンで選択する
- 総合内科転科ルートでは、平日日勤・夜間・休日のいずれでもEICU加療歴を確認する
- EICU加療歴は `EICU加療歴なし`、`EICU加療歴あり` の2択
- 平日日勤かつEICU加療歴なしは井本先生へ転科依頼
- 平日日勤かつEICU加療歴ありは内科輪番当番へ直接依頼
- 夜間・休日かつEICU加療歴なしは総合内科転科ルート
- 夜間・休日かつEICU加療歴ありは内科輪番確認後に内科診療科へ依頼
- その他はEHCU内転科対象外として転棟時転科ルートへ進む
- 泌尿器科・血管外科・心臓血管外科への夜間・休日転科は行わない

### EHCU内転科成立時の共通STEP

平日日勤：

1. 転科先へ依頼
2. 直接電話またはLINE WORKSで情報共有
3. 救急科で電子カルテ上の主科・主治医変更
4. 救急科で重症指示簿を消去
5. 転科先が一般病棟用指示簿へ書き換え
6. 転科先がドクターコール順序を記載

夜間・休日は転科先に応じてDr.call対象を分ける。

- 総合内科・消化器内科・腎臓内科・呼吸器内科・糖尿病内分泌内科：EHCU病棟番医師
- 循環器内科：Heart line当直医師
- 脳神経内科・脳神経外科：Stroke line当直医師
- 消化器外科：ER当直医師
- 救急科：ER当直医師
- 泌尿器科・血管外科・心臓血管外科：夜間・休日のEHCU内転科対象外

### EHCU看護師向けDr.callフロー

- トップ画面に必ず表示する
- まず時間帯を選択し、次に主科または転科先を選択して、誰をDr.call対象とするか確認する
- EHCU入室時とEHCU内転科後で異なる周辺作業は、判定結果の下に作業メモとして分けて表示する
- 平日日勤の総合内科は総合内科医師・NP
- 夜間休日の総合内科はEHCU病棟番医師
- 救急科主科はER当直医師
- 夜間休日の循環器内科はHeart line当直医師
- 夜間休日の脳神経内科・脳神経外科はStroke line当直医師
- その他はEHCU看護師向けDr.callフローではなく転棟時転科ルートを参照する

### 転棟時転科ルート

- 内因性疾患は内科輪番の順番通り振り分け
- 外科系・マイナー疾患はそれぞれの科へ相談
- 転棟可能となった時点で、転棟後主科へ連絡
- 直接電話またはLINE WORKSで情報共有
- 夜間・休日の転棟は原則控える
